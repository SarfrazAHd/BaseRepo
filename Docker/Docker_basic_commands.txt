							Basic Commands for docker 



you can the get  description of  all commands by typing - " Docker --help "


1 -	 docker -v or docker --version    				# it will  tell you the description of docker version

2 -	docker version 					#  Description about docker engine 

3 -	docker login 								# Login to Docker hub repository

4 -	docker run  " image id "						#  it will run local image present in your docker system

5 -	docker image ls  or    docker image -a    				# List out the all presented image in our system. 

6 - 	docker image -q								# It will return you only image id of every image

7 -    	docker ps 								# ps for process- it will return  only Running 												containers

8  -	docker ps -a 								#  it will list out all container both running and non-											running.

9 - 	docker exec "container id/name"						# this will execute your docker container

10  -	docker start  " container id /Name "					# it will start a  container	

11 -	docker stop" container id /Name "					# it will stop a running acontainer		

12  -	docker rm ''Container id  "						# Removing one or more containers

13  -	dcocker rmi  " image  id "						# Removing one or more docker images
	
14  -   dock build  "give name to your image " " location of docker file"	# It will build your custom image from you 													docker file
 
15 - 	docker run --name "give name to your cotainer" -it   "images"		# Running a docker images like ubnut or something else

16 -	docker run -p 1010 (port) : 80 nginx (image )				# Exposing port or deploying images the browser

17 - 	docker continer create "ur_container_name"				# this commmand will create a docker container

18 -    docker container exec -it "container id/name"				# this command will execute you docker container in 											intercative mode	

19 - 	docker volume  ls  							# it will list out all docker volume

20 - 	docker volume create	"Volume name"					# ti will create a docker volume

21 - 	docker volume rm 							# will delete one volume

22 - 	docker volume prune							# it will delete the all created volumes
